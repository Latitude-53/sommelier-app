# Changelog

Все заметные изменения проекта фиксируются здесь.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.0.2] — 2025-06-25

### Changed
- **Структура 8→7 осей** — параметр «Алкоголь» убран из радара и математики (information leak). Алкоголь остаётся в метаданных (ABV).
- **Симметрия 7/7** — теперь 7 параметров структуры (язык) + 7 кластеров ароматики (нос) = 14 осей радара.
- **«Солёное/Умами» → «Солёность/Минеральность»** — более понятный сенсорный маркер.
- **Математика** — знаменатель 32→28 (7×4) во всех 5 функциях сравнения. Шкала стала честнее: 25-100% вместо 50-100%.

### Fixed
- **Compare радар** — точки показывали номер напитка (1,2,3) вместо значения параметра. Теперь показывают значение (1-5).
- **Все отсылки к «8 параметров»** — обновлены на 7 во вкладках, README, комментариях.
- **DISH_RULES** — убран `alcohol` из 29 target-профилей.
- **Quiz scores** — убран `alcohol` из начальных значений.

### Added
- **8 оранжевых/натуральных вин** — УППА, Alma Valley, Рамато, Glou-Glou, Pét-Nat, Domaine Glinavos, Piquentum, Pheasant's Tears. Всего напитков: 263.

## [1.0.1] — 2025-06-25

### Fixed
- **Математика % схожести** — знаменатель `40→32` и `35→28` в 5 функциях. Шкала теперь честная (25-100% вместо 50-100%).
- **Compare** — `compare_list: []` вместо хардкод-демо. Empty-state при первом открытии.
- **Browse фильтры** — `sake`/`cider` переведены на «Саке»/«Сидр».
- **Опечатки в DISH_RULES** — `Matchingать` → `соответствовать`.
- **Гибридные символы** — `Тrou` → `Trou`, `Дouro` → `Douro`, `квеvri` → `квеври`, `ханami` → `ханами`.

### Added
- **8 оранжевых/натуральных вин** (в этом патче или предыдущем).

## [1.0.0] — 2025-06-24

### Первый релиз 🍷

- 255 напитков в 8 категориях (вино, пиво, крепкое, саке, сидр, медовуха, кофе, чай)
- 10 вкладок: Дерево, Я в тупике, Опиши, Слепая, База, По ноте, К еде, Сравнить, Словарь, Заметки
- 232 термина в словаре с алфавитной навигацией
- 29 блюд с классикой + математикой паринга
- 4 режима слепой дегустации
- Интерактивный радар с draggable точками (вкладка «Опиши»)
- Свайп-навигация с авто-скроллом хедера
- Тёмная/светлая тема, звук (Web Audio API), вибрация
- Back button Android (модалка → Дерево → выход)
- Safe-area для вырезов/чёлки
- Иконка приложения (бокал + радар)
- Скрипты сборки APK (build-apk.bat, install-icon.bat, setup-jdk.bat)
- Админка с экспортом/импортом датасета
- Полностью офлайн, 0 внешних запросов
