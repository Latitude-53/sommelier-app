# Быстрый старт (3 минуты)

## Вариант 1: Просто открыть в браузере

1. Открой файл `www/index.html` в любом браузере (Chrome/Safari/Firefox)
2. Готово! Приложение работает офлайн.

## Вариант 2: Редактировать датасет через админку (рекомендуется)

1. Открой `www/admin.html` в **Chrome или Edge** (нужен File System Access API)
2. Нажми "📁 Открыть папку data/" → выбери папку `data/` из проекта
3. Редактируй напитки/термины/режимы через удобный интерфейс:
   - Слева — список элементов с поиском
   - Справа — форма с ползунками для вкусового профиля, тегами, описанием
   - Кнопки: Добавить / Дублировать / Удалить
4. Нажми "💾 Сохранить всё" — изменения запишутся прямо в JSON-файлы
5. Запусти `python3 scripts/build.py` чтобы пересобрать HTML

**Преимущества админки:**
- Ползунки для вкусового профиля (1-5) вместо ручного JSON
- Теги добавляются кнопкой, не через запятые
- Видишь все поля напитка на одном экране
- Поиск по названию
- Валидация JSON при редактировании сложных полей

## Вариант 3: Редактировать JSON вручную

1. Открой любой файл в папке `data/` в текстовом редакторе (VS Code, Sublime, Notepad++)
2. Внеси изменения (например, добавь новый напиток в `drinks.json`)
3. Запусти в терминале:
   ```bash
   python3 scripts/build.py
   ```
4. Обнови `www/index.html` в браузере — изменения появились

## Вариант 4: Собрать Android-аппку

**Требования:** Node.js 18+, Android Studio, JDK 17+

1. Установи зависимости:
   ```bash
   npm install
   ```
2. Инициализируй Android:
   ```bash
   npm run android:init
   ```
3. Открой в Android Studio:
   ```bash
   npm run android:open
   ```
4. Нажми ▶️ Run в Android Studio → APK установится на телефон/эмулятор

**Для публикации в Google Play:**
```bash
cd android && ./gradlew bundleRelease
# → AAB файл появится в android/app/build/outputs/bundle/release/
```
Загрузи AAB через https://play.google.com/console ($25 один раз).

## Частые вопросы

**Q: Как добавить новый напиток?**
A: Открой `data/drinks.json`, скопируй существующий объект, измени поля, сохрани, запусти `python3 scripts/build.py`.

**Q: Как добавить новый режим слепой?**
A: Открой `data/blind_modes.json`, добавь новый объект с `id`, `title`, `show_radar`, `show_tags`, `show_hint`. Запусти build.

**Q: Заметки сохраняются?**
A: Да, в localStorage браузера. При Android-сборке можно добавить синхронизацию с сервером.

**Q: Работает офлайн?**
A: Да, полностью. Все данные встроены в HTML.

**Q: Можно ли на iOS?**
A: Да, через Capacitor (нужен macOS + Xcode). Аналогично Android, но `npm run ios:init`.
